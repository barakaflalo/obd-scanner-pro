# AutoDiag Pro — אבחון רכב 🔧

## העלאה ל-GitHub Pages:

### שלב 1: צור repository חדש
1. היכנס ל-GitHub שלך
2. לחץ "New repository"
3. שם: `autodiag-pro`
4. Public ✓
5. לחץ "Create repository"

### שלב 2: העלה את הקבצים
1. לחץ "uploading an existing file"
2. גרור את כל הקבצים מתיקיית dist (לא את התיקייה עצמה, את מה שבתוכה)
3. לחץ "Commit changes"

### שלב 3: הפעל GitHub Pages
1. לך ל-Settings → Pages
2. Source: "Deploy from a branch"
3. Branch: main, / (root)
4. לחץ Save
5. חכה 2-3 דקות

### שלב 4: הכתובת שלך
`https://barakaflalo.github.io/autodiag-pro/`

### שלב 5: התקנה כאפליקציה
- באנדרואיד Chrome: תפריט ⋮ → "הוסף למסך הבית"
- במחשב Chrome: אייקון התקנה בשורת הכתובת
