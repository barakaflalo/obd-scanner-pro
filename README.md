# 🔧 OBD Scanner Pro

Professional OBD-II car diagnostic scanner web app.

**Created by Barak Aflalo · [AppNest](https://barakaflalo.github.io/appnest)**

## Features

- 🔗 **4 Connection Methods** — Bluetooth BLE, WiFi, USB Serial, Demo
- 📊 **Live Dashboard** — RPM, Speed, Temp gauges + real-time graphs
- ❌ **Fault Codes** — Read/clear DTCs with 200+ code database
- ✅ **Readiness Monitors** — Emissions test status
- 📸 **Freeze Frame** — Sensor snapshot at fault moment
- 🔍 **ECU Scan** — Discover all 17+ vehicle computers
- 📊 **Odometer Comparison** — Detect km rollback across ECUs
- 🏁 **0-100 Timer** — Performance measurement
- 🗺️ **Trip Computer** — Distance, time, average speed
- ⏺ **Data Recording** — Record + export to CSV
- 🔔 **Smart Alerts** — Temperature, voltage, RPM warnings
- 🔧 **Tools** — Ohm calculator, OBD pinout, sensor table, wire colors
- 🌐 **5 Languages** — Hebrew, English, Russian, Spanish, Arabic
- 🖨️ **Report + Share** — Copy report, WhatsApp sharing
- 📱 **PWA** — Install as mobile app, works offline
- ☀️🌙 **Light/Dark Theme**

## Installation

### As a website:
Just visit: `https://YOUR-USERNAME.github.io/obd-scanner-pro/`

### As a mobile app:
1. Open in Chrome (Android) or Safari (iOS)
2. Menu → "Add to Home Screen" / "Install App"
3. The app icon appears on your home screen!

## Required Hardware

- **ELM327 BLE** adapter (NOT Classic Bluetooth!)
- Recommended: Vgate iCar Pro BLE, OBDLink MX+
- ⚠️ Delphi/Autocom adapters NOT supported (proprietary protocol)

## How to Deploy to GitHub Pages

1. Create a new repository on GitHub
2. Upload all files from this folder
3. Go to Settings → Pages → Source: "Deploy from branch" → main → Save
4. Wait 2-3 minutes, your app is live!
